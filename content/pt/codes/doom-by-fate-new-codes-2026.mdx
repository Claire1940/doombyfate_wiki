## Solução de Problemas com Códigos

Se você descobrir que um código não está funcionando, existem vários motivos comuns para que isso ocorra. Antes de desistir, verifique o seguinte:

1.  **Expiração**: A maioria dos códigos tem uma data de validade oculta. Se um código foi lançado para um feriado (como o Ano Novo), provavelmente não funcionará em abril.
2.  **Uso Único**: Você só pode resgatar cada código exclusivo uma vez por conta. Se você já o resgatou, o jogo geralmente exibirá uma mensagem de "Código já usado".
3.  **Erros de Digitação**: Espaços extras no início ou no final de um código (frequentemente causados por copiar e colar) farão com que o sistema o rejeite.
4.  **Versão do Servidor**: Ocasionalmente, novos códigos só funcionam em servidores atualizados. Se uma atualização acabou de ser lançada, tente entrar em uma nova instância de servidor.
