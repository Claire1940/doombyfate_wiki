## Perintah Penting Server Privat Doom By Fate

Dalam build 2026 saat ini, daftar perintah difokuskan pada utilitas dan moderasi. Karena Doom By Fate mengandalkan keseimbangan yang tepat antara delapan Penyintas dan satu Pembunuh, perintah-perintah ini membantu mengatur ulang status permainan jika terjadi bug atau jika ada pemain yang terputus secara prematur.

### Utilitas Administratif
Perintah-perintah ini adalah hal mendasar bagi setiap tuan rumah. Mereka memastikan bahwa pertandingan tetap kompetitif dan semua orang mengikuti aturan yang ditetapkan oleh pemilik server.

*   **!kick [NamaPengguna]**: Segera mengeluarkan pemain dari sesi saat ini. Gunakan ini untuk pemain yang menolak mengikuti panduan roleplay atau pengujian.
*   **!map [NamaPeta]**: Memaksa server untuk memuat lokasi tertentu. Ini sangat berguna untuk mempelajari POI (*Points of Interest*) pada peta baru yang ditambahkan dalam Update Jeffy.
*   **!reset**: Memulai ulang putaran saat ini. Jika Pembunuh terjebak atau Penyintas mengalami *glitch* menembus geometri, perintah ini adalah cara tercepat untuk kembali beraksi.

### Konfigurasi Pertandingan
Menyesuaikan tingkat kesulitan atau alur pertandingan adalah alasan utama menggunakan server privat. Meskipun inti permainan adalah 8v1, beberapa tuan rumah lebih suka menguji skenario 4v1 atau mode petak umpet khusus "Hanya Pembunuh".

| Tipe Perintah | Tujuan | Contoh Sintaks |
| :--- | :--- | :--- |
| **Batas Waktu** | Menyesuaikan durasi putaran. | `!time 600` |
| **Penetapan Peran** | Memaksa pemain tertentu ke dalam peran Pembunuh. | `!setkiller [Nama]` |
| **Mod Kecepatan** | Menyesuaikan kecepatan gerakan untuk pengujian (Hanya Alpha). | `!speed [Nama] [Nilai]` |
